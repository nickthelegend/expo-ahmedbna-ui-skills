# Getting Started with Expo UI Skills

## 1. Installation
Install the `agent-skills` CLI and add this repository:
```bash
npx skills add nickthelegend/expo-ahmedbna-ui-skills
```

## 2. Peer Dependencies
Make sure you have the following installed in your Expo project:
- `react-native-reanimated`
- `lucide-react-native`
- `expo-haptics`
- `expo-audio` (for audio components)

## 3. Importing Components
Once added, you can import components directly from your UI folder:
```tsx
import { Button } from '@/components/ui/button';
```
