# Instructions for AI Agents

This repository is designed to be consumed by advanced AI coding agents (Claude, GPT-4, etc.).

## 🤖 How to use this collection
1. **Explore**: Scan `index.json` to see all available UI skills.
2. **Learn**: Read the `SKILL.md` in any skill directory to understand the implementation and best practices.
3. **Implement**: Copy the code from the `references/` folder to your project.
4. **Optimize**: Use the provided AI prompts at the bottom of each `SKILL.md` to trigger advanced reviews.

## 🛠️ Tooling Integration
If you have the `agent-skills` CLI installed:
```bash
npx skills add nickthelegend/expo-ahmedbna-ui-skills
```
This will unpack all skills into your local environment for easy access.
