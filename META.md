# Expo UI Skills Maintenance Metadata

This collection is maintained to ensure the highest quality of React Native and Expo components.

## 1. Quality Standards
- All skills must include a `references/` folder with the full source code.
- Components must be verified for both iOS and Android compatibility.
- Accessibility (Aria-ready) is a mandatory requirement for all UI components.

## 2. Structural Integrity
- The `index.js` and `index.json` must be updated after every new skill addition.
- `SKILL.md` must contain an "AI Agent Prompt" to facilitate LLM-based development.

## 3. Versioning
- **v1.x.x**: Initial components and charts.
- **v2.x.x**: Integration with Reanimated 3 and advanced state management.
