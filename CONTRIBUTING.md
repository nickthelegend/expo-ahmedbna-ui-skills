# Contributing to Expo UI Skills

Thank you for your interest in improving the Expo UI Skills collection!

## How to Contribute
1. **Request a Skill**: If a component from `ahmedbna/ui` is missing, open an issue.
2. **Improve Documentation**: Found a typo or a better example? Submit a PR.
3. **Add Examples**: We love new high-level templates in the `EXAMPLES/` directory.

## Development Workflow
1. Clone the repository.
2. Add a new skill directory in `skills/expo-[name]`.
3. Create a `SKILL.md` following the existing pattern.
4. Run the index generation script to update `index.js` and `index.json`.
5. Submit a Pull Request.

## Code of Conduct
Please be respectful and professional in all interactions.
